Nasıl Çalıştırılır?

İndirdiğiniz whatsbotg.exe dosyasını istediğiniz bir klasöre koyun ve komut istemcisinden (cmd/PowerShell) çalıştırın.

REST API modunda çalıştırmak için:

whatsbotg.exe rest

MCP (Model Context Protocol) sunucusu olarak çalıştırmak için:

whatsbotg.exe mcp
Not: Aynı anda hem REST hem de MCP modunda çalıştıramazsınız, ikisinden birini seçmeniz gerekir. Ayrıca medya işlemleri için FFmpeg'in sisteminizde kurulu olması gerekir.
